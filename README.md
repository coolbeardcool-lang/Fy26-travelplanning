# FY26 Travel Planning

GitHub Pages site for the FY26 Kansai outing plan.

## Contents
- `index.html`: main page with trip board, map, routes, and spot cards
- `style.css`: layout and card styles
- `data.js`: itinerary, points, and route data
- `app.js`: Leaflet map and interactive filtering logic
- `PAGES_SETUP.md`: how to enable GitHub Pages
